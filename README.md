# CarND-Kidnapped-Vehicle-P3
Udacity Self-Driving Car Nanodegree - Kidnapped Vehicle project

## Running the Code
Execute the following in the top directory of the project:

1. ./clean.sh
2. ./build.sh
3. ./run.sh
